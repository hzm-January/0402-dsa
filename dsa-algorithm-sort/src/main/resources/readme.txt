如何分析一个排序算法？

1. 排序算法执行效率
1.1 时间复杂度分析：最好情况时间复杂度；最坏情况时间复杂度；平均情况时间复杂度
1.2 时间复杂度的系数、常数、低阶
1.3 比较次数和交换(或移动)次数

2. 排序算法内存消耗(空间复杂度分析)
3. 排序算法稳定性--在排序过程中，不会改变相等元素的先后顺序




相关概念
    有序元素对：a[i] <= a[j]，如果i < j
    逆序元素对：a[i] > a[j]，如果i < j

    有序度：数组中具有有序关系的元素对个数；
    逆序度：数组中具有逆序关系的元素对个数；
    满有序度：完全有序数组的有序度
    逆序度 = 满有序度-有序度


    数组可分为两个区间：已排序区间，未排序区间
